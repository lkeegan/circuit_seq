# CircuitSeq Server

A work-in-progress Flask prototype of the back end.

## Installation

```pycon
pip install git+https://github.com/ssciwr/circuit_seq_server.git
```

## Use

To start a local development server for testing purposes:

```bash
python -m circuit_seq_server
```
